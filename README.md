# Chat App with Sails.js - Server

### Clone this repo:
```
> git clone https://github.com/AbhishekDutt/ReduxChatServer.git
```

### Install dependencies:
```
> cd ReduxChatServer
> npm install
```

### Run:
```
> sails lift
```
